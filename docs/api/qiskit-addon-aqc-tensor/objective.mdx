# Objective Function

Code for building and evaluating objective functions used for AQC parameter optimization.

Currently, this module provides the simplest possible objective function, [`OneMinusFidelity`](objective-one-minus-fidelity "qiskit_addon_aqc_tensor.objective.OneMinusFidelity").
