# qiskit.pulse.ShiftPhase.shift

`ShiftPhase.shift(time, name=None)`

**Parameters**

*   **time** (`int`) – Time to shift by
*   **name** (`Optional`\[`str`]) – Name of the new schedule. Defaults to name of self

**Returns**

The shifted schedule.
