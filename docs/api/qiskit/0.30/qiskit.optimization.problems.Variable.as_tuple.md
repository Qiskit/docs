# qiskit.optimization.problems.Variable.as\_tuple

`Variable.as_tuple()`

Returns a tuple corresponding to this variable.

**Return type**

`Tuple`\[`str`, `Union`\[`float`, `int`], `Union`\[`float`, `int`], `VarType`]
