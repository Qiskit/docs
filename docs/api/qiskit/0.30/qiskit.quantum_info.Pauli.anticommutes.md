# qiskit.quantum\_info.Pauli.anticommutes

`Pauli.anticommutes(other, qargs=None)`

Return True if other Pauli anticommutes with self.

**Parameters**

*   **other** ([*Pauli*](qiskit.quantum_info.Pauli#qiskit.quantum_info.Pauli "qiskit.quantum_info.Pauli")) – another Pauli operator.
*   **qargs** (*list*) – qubits to apply dot product on (default: None).

**Returns**

True if Pauli’s anticommute, False if they commute.

**Return type**

bool
