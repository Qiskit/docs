# qiskit.quantum\_info.StabilizerState.tensor

`StabilizerState.tensor(other)`

**other** ([*StabilizerState*](qiskit.quantum_info.StabilizerState#qiskit.quantum_info.StabilizerState "qiskit.quantum_info.StabilizerState")) – a stabilizer state object.

**Returns**

the tensor product operator self ⊗ other.

**Return type**

[StabilizerState](qiskit.quantum_info.StabilizerState#qiskit.quantum_info.StabilizerState "qiskit.quantum_info.StabilizerState")

**Raises**

**QiskitError** – if other is not a StabilizerState.
