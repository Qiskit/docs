# to\_matrix

<span id="qiskit.quantum_info.Pauli.to_matrix" />

`Pauli.to_matrix()`

Convert Pauli to a matrix representation.

Order is q\_\{n-1} …. q\_0, i.e., \$P\_\{n-1} otimes … P\_0\$

**Returns**

a matrix that represents the pauli.

**Return type**

numpy.array
