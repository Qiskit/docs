# check\_xtalk\_dependency

<span id="qiskit.transpiler.passes.CrosstalkAdaptiveSchedule.check_xtalk_dependency" />

`CrosstalkAdaptiveSchedule.check_xtalk_dependency(t_1, t_2)`
