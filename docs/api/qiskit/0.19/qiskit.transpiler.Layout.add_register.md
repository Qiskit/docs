# add\_register

<span id="qiskit.transpiler.Layout.add_register" />

`Layout.add_register(reg)`

Adds at the end physical\_qubits that map each bit in reg.

**Parameters**

**reg** (*Register*) – A (qu)bit Register. For example, QuantumRegister(3, ‘qr’).
