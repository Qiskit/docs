  Clifford synthesis plugin based on the Bravyi-Maslov method to synthesize Cliffords into layers.

  This plugin name is :`clifford.layers` which can be used as the key on an [`HLSConfig`](transpiler-passes-hls-config "qiskit.transpiler.passes.HLSConfig") object to use this method with [`HighLevelSynthesis`](transpiler-passes-high-level-synthesis "qiskit.transpiler.passes.HighLevelSynthesis").
