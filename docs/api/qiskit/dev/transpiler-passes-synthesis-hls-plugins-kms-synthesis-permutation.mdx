  The permutation synthesis plugin based on the Kutin, Moulton, Smithline method.

  This plugin name is :`permutation.kms` which can be used as the key on an [`HLSConfig`](transpiler-passes-hls-config "qiskit.transpiler.passes.HLSConfig") object to use this method with [`HighLevelSynthesis`](transpiler-passes-high-level-synthesis "qiskit.transpiler.passes.HighLevelSynthesis").
