  A QFT-based adder with a carry-in and a carry-out bit.

  This plugin name is:`HalfAdder.qft_d00` which can be used as the key on an [`HLSConfig`](transpiler-passes-hls-config "qiskit.transpiler.passes.HLSConfig") object to use this method with [`HighLevelSynthesis`](transpiler-passes-high-level-synthesis "qiskit.transpiler.passes.HighLevelSynthesis").
