# Initialize account using REST APIs

Qiskit Runtime `Sampler` and `Estimator` are managed services which require you to initialize your account. After initialization, you can select the IBM Quantum Compute resource of your choice to run your calculations on. 

### Setup temporary access token 

Obtain a temporary Access token by supplying the API token from the IBM Quantum Platform. This is especially useful if you would like control over tokens, such as token invalidation. Alternatively, you can work directly with your IBM Quantum Platform API token.

<Admonition type="note">
    The code snippets below assume that the API Token from the IBM Quantum Platform is stored in a filed named `token`.
</Admonition>

```python
import requests

with open('token') as file:
    token=file.read()

url = 'https://auth.quantum-computing.ibm.com/api/users/loginWithToken'
input={'apiToken': token}
auth_response = requests.post(url, json=input)
auth_id=auth_response.json()['id']
```

### View available backends

Let's view your available backends

```python
url_backends = 'https://api.quantum-computing.ibm.com/runtime/backends'
headers = {'Content-Type': 'application/json',
            'x-access-token':auth_id}

backends_response = requests.get(url_backends, headers=headers)

print(backends_response.json()['devices'][:5],"...")
```

### Transpile and execute circuits

You can then transpile circuits using the [Qiskit transpiler service](rest-api-transpilation) and run them using either the [Sampler](rest-api-sampler) Primitive or the [Estimator](rest-api-estimator) Primitive.   

After your experiments are complete, you can proceed to invalidate your token followed by testing it's invalidation.


### Invalidate Token and test Token Invalidation

```python
logout_url = 'https://auth.quantum-computing.ibm.com/api/users/logout'
headers = {'x-access-token':auth_id}
logout_response = requests.post(logout_url, headers=headers)
print("response ok?:",logout_response.ok,logout_response.text)
```

This should yield an error (Error 401) once access token is invalidated.

```python
logout_url = 'https://auth.quantum-computing.ibm.com/api/users/logout'
headers = {'x-access-token':auth_id}
logout_response = requests.post(logout_url, headers=headers)

if logout_response.status_code == 200:
    job_id = logout_response.json().get('id')
    print("Job created:",logout_response.text)
elif logout_response.status_code == 401:
    print("invalid credentials. Access token should be successfully invalidated.")
else:
    print(logout_response.text,"\n")
    print(f"Error: {logout_response.status_code}")
```
Output
```text
invalid credentials. Access token should be successfully invalidated.
```
